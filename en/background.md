# Background - 臺灣健康檢查資料交換實作指引 (Taiwan Health Assessment Implementation Guide, TWHA IG) v0.1.0

## Background

# 計畫背景與法規遵循 (Background & Compliance)

## 1. 計畫背景與痛點 (Project Background & Pain Points)

近年衛生福利部積極推動數位健康基礎建設，然而國內健康檢查資料目前仍普遍存在下列問題，使其難以納入上述數位基礎建設：

* **資料格式不一致**：不同醫院與健檢機構分別使用 PDF、Excel、CSV 或院內私有資料庫等格式儲存健檢結果，缺乏共通結構。
* **無法跨院交換**：受檢者於各級醫療機構或健檢診所完成檢查後，結果難以透過標準化格式跨機構交換，亦無法回饋至「健康存摺」。
* **政府資料整合困難**：勞工健康檢查、成人預防保健、企業自費健檢等各類資料缺乏一致標準，造成中央主管機關（如衛福部、國健署、勞動部職安署）彙整、監測與政策制定之困難。

-------

## 2. 計畫定位與目標 (Project Positioning & Objectives)

為解決上述問題，本計畫建立「臺灣健康檢查資料交換實作指引（TWHA IG）」，作為全國健康檢查之 FHIR 交換標準，其實體目標包括：

1. **建立全國共通健康檢查核心資料集**：定義 Core Assessment Dataset，做為共通核心之基準。
1. **建立 FHIR 資料交換架構**：以 Observation、DiagnosticReport、Composition、Bundle 等資源為核心，定義標準化 FHIR 資料建模方式。
1. **建立領域特定交換標準**：涵蓋《勞工健康保護規則》之勞工健康檢查標準，以及國民健康署之成人預防保健服務資料標準。
1. **推動跨機構交換示範**：展示由醫療機構產生 FHIR Exchange Package、上傳至平台並成功解析之完整資料交換流程。

-------

## 3. 法規與政策依據 (Regulatory & Policy Context)

### 3.1 勞工健康檢查與服務紀錄

本指引遵循我國職業安全衛生與勞動保護法規之要求：

* **《職業安全衛生法》第 20 條**：雇主於僱用勞工時，應實施「體格檢查」；對在職勞工應實施定期「一般健康檢查」；對從事特別危害健康作業之勞工，應實施定期「特殊健康檢查」。
* **《勞工健康保護規則》**：本指引勞工領域之直接法規依據。 
* **第 14 條**：明定雇主對辦理勞工健康服務時，應依「附表八」項目完整記錄，並至少保存七年。
* **第 16、17 條**：一般體格與健康檢查之項目與週期如「附表九」。
* **第 18 條**：特別危害作業特殊健檢之項目如「附表十」。
* **第 21、23 條**：健康管理分級判定（一至四級）與後續適性配工或追蹤措施。
 

### 3.2 成人預防保健服務

* **國民健康署成人預防保健服務**：旨在為中年及老年國民提供早期的慢性病篩檢與健康風險評估。服務對象包含： 
* 40歲至64歲：每3年一次。
* 65歲以上、55歲以上原住民、35歲以上罹患小兒麻痺者：每年一次。 透過本指引所制定的問卷（QuestionnaireResponse）與檢驗指標（LOINC 碼），將其納入全國統一的 FHIR 資料交換體系中。

## 4. 資料保存年限要求 (Data Retention Policies)

為確保職業史、生活習慣與疾病因果關係之追溯，相關健康檢查紀錄之保存年限如下：

| | | |
| :--- | :--- | :--- |
| **一般體格/健康檢查** | 至少保存**7 年** | 適用於一般受檢者及在職勞工之基本紀錄。 |
| **特殊體格/健康檢查** | 至少保存**10 年** | 適用於噪音、鉛、粉塵等大多數特別危害作業。 |
| **特定暴露作業健檢** | 至少保存**30 年** | 適用於石綿、游離輻射、聯苯胺等致癌或具長期潛伏期之作業。 |
| **臨場服務執行紀錄** | 至少保存**7 年** | 附表八之臨場健康服務紀錄。 |

## 5. 技術標準遵循 (Technical Standards Compliance)

本實作指引之編製遵循**衛生福利部資訊處**對於數位醫療平臺標準化之要求：

* **與 TW Core IG 互操作**：繼承中華民國 HL7 FHIR 國家核心標準 `TW Core IG v1.0.0`，確保本指引所產生之資料能順利整合至電子病歷 (EMR) 系統與次世代數位醫療平臺。
* **術語服務 (Terminology Service)**：檢驗與測量項目全面採用國際標準 **LOINC** 與 **SNOMED CT**，確保臨床語義的一致性與互操作性。
